Bienvenidos

ABC INVENTORY: Hola Bienvenidos a nuestro menu como pueden ver tienen la opciones para ver nuestros tipos de procesos que llevamos acabo aparte del inventario de nuestro negocio.

  • Solicitar un producto.
  • Registrar un producto.
  • Modificar plazo de entrega
  • Inventario-Eliminar un Producto
  • Seleccionar Metodo de Pago

Solicitar un Producto

El gerente tiene esta responsabilidad ya que tiene que ver que productos se compran y cuales no se compran para no solicitarlos en la siguiente entrega esto tambien los empleados pueden comentarlo con el gerente.

Resgistrar un producto

El Resgistrar un producto lo marcamos como que se hizo un pedido de ese producto o solo se resgitra como producto lleno para que no solicitemos mas y se llamen a mandar otros mas se hace un inventario de que productos son los mas vendidos y lo que menos se vende en la tienda.

Modificar plazo de entrega

Modificar plazo de entrega se refiere cuando el producto lo nececitamos cambiar por razones que afenten a la entrega del producto

Inventario-Eliminar un Producto

Lo priemero se realiza el inventario para ver que productos se hizo mas gasto y cuales no para despues eliminarlos de la tienda o no pedir el producto despues para no realizar gastos inecesarios.

Seleccionar Metodo de Pago

Este el tipo de pago que nosotros podemos realizar desde debito a credito. Ponemos nuestros datos que nos pidan.
